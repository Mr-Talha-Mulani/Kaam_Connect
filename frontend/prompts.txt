Global UI Prompt
Use Next.js (App Router) with React and Tailwind CSS.
Design should be clean, modern, and mobile-first.
Use soft colors, rounded corners, and minimal UI.
Focus on accessibility and simplicity for non-technical users.
Avoid clutter. Use large buttons and clear labels.

1. Landing Page (/app/page.js)
Create a landing page for a platform called "KaamConnect".

Purpose:
Connect job seekers with employers for blue-collar jobs.

Sections:
- Hero section with title:
  "Find Work. Hire Workers. Instantly."
- Subtitle explaining the platform
- Two CTA buttons:
  - "Find a Job"
  - "Hire Workers"
- Job categories grid:
  (Driver, Helper, Delivery, Cleaner, Store Assistant)
- How it works (3 steps):
  - Sign up
  - Complete profile
  - Connect directly
- Footer

Design:
- Clean, modern, mobile-first
- Use icons for categories
- Big buttons
